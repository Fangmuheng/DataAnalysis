#这里计算的是每个岗位数据的平均薪资，取平均。
def salary_count(x):
    x_min,x_max = x.split('-')  #获取-两边的最大最小值
    x_min = int(x_min)*1000
    x_max = int(x_max.replace('K','000')) #截取字符串，把k改成000，将字符串转化为数字
    if x_min == 0:
        x_min = 1    #除数不能为0
    return np.mean([x_min,x_max])

从上图可以看出，除了不限学历水平以外，随着经验年限的增长，薪资增长最快的是要求大专学历的岗位，最慢的是要求硕士学历的岗位。

可以猜想，经验年限要求越高，对学历的重视水平越低，属于是“工作经验更重要”。
